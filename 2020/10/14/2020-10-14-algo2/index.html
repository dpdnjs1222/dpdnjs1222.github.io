[알고리즘] 2020 카카오 신입공채 풀어보기 - 문제 2

Posted on 2020-10-14 Edited on 2021-02-14 In algorithm

문제 링크

1. 문제 설명

  • 균형잡힌 괄호 문자열 : ‘(‘와 ‘)’의 개수가 같음 ex :(()))(
  • 올바른 괄호 문자열 : ‘(‘와 ‘)’의 짝이 맞음 ex :(())()
  • 균형 잡힌 괄호 문자열을 올바른 괄호 문자열로 변환하는 방법이 주어짐

2. 문제 해결 방법

문제에 명시되어있는 변환 방법을 구현한다.
문제 곳곳에 재귀로 구현해야한다는 단서가 있다.

3. 구현 코드 및 설명

  1. “균형잡힌 괄호 문자열” 판단 - divide_even 함수
  2. “올바른 괄호 문자열” 판단 - is_correct 함수 : 괄호문자열은 스택으로 판단
  3. 괄호 방향을 뒤집기 위해 ASCII 이용 - ord, chr

괄호 방향을 뒤집어야 하는데 문자열을 뒤집는걸로 잘못 이해하여 처음에 실패하였다. 참고로 문자열 뒤집기는 [::-1]이다

4. 타 소스 비교 및 느낀 점

“균형잡힌 괄호 문자열”과 “올바른 괄호 문자열”을 c, r 두 변수로 확인한다. (문자열 앞에서 부터 ‘(‘인 경우 c += 1, ‘)’인 경우 c -= 1 한다 -> 문자 두개씩 볼 때마다 c가 0인 경우 올바른 괄호 문자열이고 c가 0보다 작은 경우 균형잡힌 괄호 문자열이 아니다)
문자열의 괄호 방향을 뒤집기 위해 람다함수와 map을 사용하였다.

항상 느끼는 것이지만 문제를 한번에 잘 이해해야 한다.
